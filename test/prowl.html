Introduction

Introduction guide to Prowl

Welcome to the ProwlJS Docs.

Prowl is a jQuery plugin which enables an extensively customisable modal to be implemented on any site.

Get started with the setup below

Setup

Installing and setting up Prowl

Download the latest distribution files from the repo.

Include the CSS files in the head of your page (prowl.css is required unless you know what you're doing):

										<link rel="stylesheet" href="../dist/prowl.css">
<link rel="stylesheet" href="../dist/prowl_theme.css">

Include the JS just before 

										<script src="../dist/prowl.js"></script>

Create the basic modal

										<a class="open">Open the modal</a>
<div class="prowl">
	<div class="prowl-overlay">
	<div class="prowl-modal">
		<h2>It's a modal!</h2>
		<div class="button toggle">
				Close
		</div>
	</div>
	</div>
</div>

Initiate it based on what toggle you want to trigger the modal (we use the class .open from the html above)

										$('.open').prowl()

Options

Available options for the Prowl object

You can pass additional options to Prowl

container 
										Default: .prowl
toggleClass

Add what class name will be appended to the toggle element on state change.

										Default: .prowl-toggle
overlay

The element class of the overlay of the modal.

										Default: .prowl-overlay
modal

The element class of the modal itself.

										Default: .prowl-modal
background

Override the color of the overlay background, formatted in hex, rgba or rgb.

										Default: rgba(149, 155, 160, 0.57)
animate

Choose animation of how the modal enters and exits. Options are fade, reveal, swash and drop.

										Default: fade
duration

The speed of the animation in ms.

										Default: 200
closeOnEscape

If set to true, the modal closes when the escape key is pressed.

										Default: true
state

Override the initial state of the modal. Options are open, closed

										Default: closed

Methods

Types of methods exposed

Assign the instance to a variable and take advantage of methods:

										let inst = $('.open').prowl()
open()

Opens the modal.

										inst.open()
close()

Closes the modal.

										inst.close()
getState()

Gets the current state of the modal.

										inst.getState()
toggle()

Toggles the current visibility of the modal.

										inst.toggle()
validColor(color)

Pass a color and check if it is valid for the overlay background.

										inst.validColor("#000000")

Events

Perform actions when events are triggered

Assign functions to inner events on state transitions

onOpen

Calls the passed function when the modal opens.

										let inst = $('.click').prowl({
	onOpen: () => {
		console.log('Modal has opened')
	}
})
onClose

Calls the passed function when the modal closes.

										let inst = $('.click').prowl({
	onClose: () => {
		console.log('Modal has closed')
	}
})
onOpening

Calls the passed function when the modal is opening (the time between the trigger to the time the animation finishes).

										let inst = $('.click').prowl({
	onOpening: () => {
		console.log('Modal is opening')
	}
})
onClosing

Calls the passed function when the modal is closing (the time between the trigger to the time the animation finishes).

										let inst = $('.click').prowl({
	onClosing: () => {
		console.log('Modal is closing')
	}
})
